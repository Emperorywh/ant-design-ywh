## Development

```bash
# 第一步下载依赖：install dependencies
$ npm install

# 第二步启动项目：develop library by docs demo
$ npm start

# build library source code
$ npm run build

# build library source code in watch mode
$ npm run build:watch

# build docs
$ npm run docs:build

# check your project for potential problems
$ npm run doctor
```
